# SHADOW // Qwik App

Qwik tabanlı, modüler ve görsel olarak korunmuş storefront dönüşümü.

## Komutlar

```bash
npm install
npm run check
npm run build
npm run dev
```

## Yapı

- `src/components/*` — UI bileşenleri
- `src/data/*` — veri kaynakları
- `src/lib/*` — yardımcı fonksiyonlar ve aksiyonlar
- `src/context/*` — uygulama context'i
- `src/root.tsx` — uygulama kabuğu
